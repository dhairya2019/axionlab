<objective>
Fix page-level HTML structure for SEO: enforce single H1 per page, add canonical URLs to all pages, and update homepage meta description with brand keywords.

Purpose: Google penalizes pages with multiple H1 tags (signals topic confusion) and without canonical tags (duplicate content risk). The Nav component currently renders H1 tags for the brand name on every page, creating 2-3 H1s per page. Fixing these structural issues is the foundation that enables all other SEO work.

Output: All pages have exactly one H1, every page has a canonical URL in metadata, and the homepage description contains primary brand keywords.
</objective>
